# My Python Playground

This is my playground where I'm using it to study and practice Python.

## Installing

First, go to https://www.python.org/downloads/ and download the version for your OS that best suits for you.
Then, you can follow the installing steps clicking [here](https://docs.python.org/3/using/windows.html)

## Running

To run any of this classes, you can navigate through the project using any command line and then use 'python class.py'.
Make sure that you have installed everything that the step-by-step above mentioned.
